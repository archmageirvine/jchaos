# There is a line for each item which can possibly appear on a
# player's spell list.  The format of each line is the name of the
# Java class for the spell, its frequency of occurrence in the
# range 0 to 100 inclusive and an optional flag.  A flag value of "B"
# indicates the spell can only be obtained as a bonus; that is, either
# as an explicit bonus spell or through meditation.  The order of
# the lines is not significant.  A spell with frequency 0 will never
# appear on a spell list.  Thus, for instance, if you hate the
# cloaked creatures you can set the Cloak frequency to 0 and then
# nobody will ever get a cloak spell.
#
# Be aware that it is easy to ruin the game by making powerful spells
# like demons and hyperclone common. In my experience once you cheat
# at a game much of the original enjoyment you get from the game
# is lost.
